Contains the files related to BERT based classifier models.

~~~
./data_extractor.py        --> codes related to loading the train, val and test files depending on the training setting
./feature_generation.py    --> codes related to encoding of the given input text using BERT Tokenizer. 
./own_bert_models.         --> codes for selecting the required BERT model and loading the pretrained model for finetuning
./utils.py                 --> Miscelleneous codes required by the classifier.
~~~
